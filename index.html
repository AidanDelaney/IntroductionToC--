Step by Step

Develop a space invaders style game in C++ using SDL and a RaspberryPi. If you don't have RaspberryPi hardware, you can follow on a RaspberryPi emulator.

Strongly Architected

We use modern C++ (C++11) to interface with the SDL library, written in C. On top of this we implement a basic sprite engine and collision detection.

Learning C++ through Game Development

Environment Smoketest. Running our sample game.

Our sample game (half a game really) is on GitHub. In this video we look at checking out the source-code and building it. To do this, we take the long way round using the Linux command line. It might be easier to use a GUI, but I tend to think that this approach is good for you.

The code in this project is much more advanced than what we'll start with. It's a smoke-test, a quick-and-dirty check to see if your development environment works correctly. In the following guides, we start from the first step and, subsequently, work our way up to the example we demonstrate here.
